Doll machine

A doll machine is a hypothetical contraption first described by Jane Rieler in 2006 that takes in inorganic matter and creates a living, albeit unconscious, creature. Its successful creation, alongside the absence of a consciousness-producing variant, would challenge the standing theory that conscious and unconscious beings come into being the same way (naturally), and for the same reason (survival).

Rieler uses the term doll in multiple talks, the earliest in 2002, as an analogy for "something that resembles a conscious being, but isn't one itself." This analogy is what led her to later propose the doll machine thought experiment.

Some speculate that dolls have already been artificially created in the form of artifical intelligence, cellular automata, and so forth, while others argue that Rieler's description only lends itself to organic life.

Thought experiment

It is either the case that,

  1. A machine cannot be produced that can create conscious or unconscious life; This would imply that organic life as a whole cannot be artificially synthesized within the Universe.
  2. A machine can be produced that can create unconscious life (this is the doll machine), but not one that creates conscious life; This would suggest that conscious and unconscious life are unrelated, and that conscious beings do not arise from unconscious ones (if they did, a doll machine would also be a consciousness-producing machine).
  3. A machine can be produced that can create both conscious and unconscious life.

Rieler puts it simply: "If the tools that create life exist entirely within our Universe, then we ourself, not just random chance, should be able to create life."

The fourth permutation, that a machine can be produced that can create conscious life but not unconscious life, is disregarded by Rieler.

Implications of the divine

If a doll machine was made, and it was proven that a machine that can produce consciousness cannot be made, then there must be a god. But if conscious life can arise naturally without godly intervention, we should be able to replicate that process.
Jane Rieler
