Doll machine

A doll machine is a hypothetical contraption first described by Jane Rieler in May of 2006 that takes in inorganic matter and creates a living, albeit unconscious, creature. Its successful creation, alongside the absence of a consciousness-producing variant, would challenge the standing theory that conscious and unconscious beings come into being the same way (naturally), and for the same reason (survival).

Rieler uses the term doll in multiple talks, the earliest in 2002, as an analogy for "something that resembles a conscious being, but isn't one itself." This analogy is what led her to later propose the doll machine thought experiment.

Some speculate that dolls have already been artificially created in the form of artifical intelligence, cellular automata, and so forth, while others argue that Rieler's description only lends itself to organic life.

Implications

If a doll machine was made, and it was proven that a machine that can produce consciousness cannot be made, then there must be a god. But if conscious life can arise naturally without godly intervention, we should be able to replicate that process.
Jane Rieler
