Anomalous Entity Classification

Anomalous Entity Classification is a system that serves to better document the distribution of anomalous entities.

Class A

Class A entities are capable of reproduction and are typically integrated into the ecosystem in a similar way to animals.

Class B

Class B entities are incapable of reproduction and so their origin is typically artificial/man-made.

Class C

Entities classified under Class C have been found to, due to some property, only exist as a sole individual within the entire universe. These include beings that support a universal law (such as the Medulla with the expansion of the universe), Jane Rieler's god (as proposed in her doll machine thought experiment), and so forth.

Class CU

Similar to Class C entities, only one Class CU entity of a given type exists, the difference being its uniqueness has yet been proven to be true. The formation of this classification is from Class C and the letter U representing "Unknown."

Plus H

Plus H is a class modifier (written in the form Class C+H) that describes hypothetical anomalous entities. It maintain's the base classes description, but with the addition that they have not been directly observed but are known to exist due to their effect on their environment.
