Anomalous Entity Classification

Anomalous Entity Classification is a system that serves to better document the habitual features of anomalous entities.

Class A

can reproduce, ecosystem

Class B

can't reproduce, typically of man-made origin

Class C

only one exists

Class CU

unknown if only one exists

Class H

Class H entities are hypothetical anomalous entities that have not been directly observed but are known to exist due to their effect on their environment.
